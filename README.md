# 🦌 Desfile Palanca Negra 2026 – Quebrando Silêncio

Site oficial do Desfile de Ordem Unida do Clube de Desbravadores Palanca Negra.

**Tema:** *Quebrando Silêncio, Uma Verdade Desconhecida*  
**Data:** 28 de Junho de 2026  
**Local:** Luanda, Angola (IASD Central de Luanda)

## 📌 Sobre o Projecto

Página única interactiva que apresenta todas as informações do desfile:

- Cronograma detalhado do evento
- Rota interactiva com mapa (Leaflet)
- Lista de comissões e membros da organização
- Regras para visitantes e contactos de apoio
- Contagem decrescente para o evento
- Transmissão ao vivo (bloqueada até ao dia)
- Partilha personalizada no WhatsApp
- Tema claro/escuro com detecção automática

## 🚀 Tecnologias Utilizadas

- HTML5 semântico
- CSS3 com variáveis e animações GSAP
- JavaScript (ES6)
- Leaflet.js para mapas interactivos
- GSAP ScrollTrigger para animações de entrada
- Fontes Google (Cormorant Garamond, Outfit, Barlow Condensed)

## 🛠️ Como executar localmente

1. **Clone ou descarregue** este repositório.
2. Certifique-se de que a pasta `assets/` contém todas as imagens (logótipo, fotos dos membros, ícones).
3. Abra o ficheiro `index.html` directamente no seu browser.
